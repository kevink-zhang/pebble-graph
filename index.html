Pebbles

Click to create a node. Press e on it to add 1 to its initial value. You can also type out a number to set a value for a node, and "~" to set the value to 0. Click on two nodes to link them. Double click or press esc to unselect a node.

Press space to play/pause the simulation. Press r to clear the graph. Press s on a node to toggle sink. Press z to yeet the selected node. Select and shift click to yeet an edge between the two nodes.

There is also a presimulation mode. After creating a graph, press p. Then, you can replay the simulation with the slider.

The simulation will pause once it determines the graph is infinitely unstable. Unstable will then become true;

Graph Generation Inputs

Press t to generate a tree. Press k to generate a complete graph. Press c to generate a cycle graph. Press g to generate a grid graph. Press o to set each node value to a random number.
